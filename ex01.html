Aprendendo DOM

É uma API que permite manipularmos o HTML com JS. O documento HTML é visto como "document" e terá associado a ele vários métodos.

  1. getElementById("nome") - Pega o emlemento pelo id
  2. innerHTML - Inserir ou mudar conteúdo
  3. Mudar estilo
  4. classList - Insere uma classe ao elemento HTML
    • .add - Adicioma uma classe ao elemento
    • .remove - Remove uma classe do elemento.
    • .toggle - Insere ou remove a classe se tiver o elemento.
  5. createElement - Cria elementos.
  6. querySelectorAll - Seleciona vários elementos.
